# Axis Connection Map — 18축 간 연결 맵

전략 도출 시 반드시 참조. 한 축의 판단이 다른 축에 미치는 영향.

## 연결표

| 선행 | → 후행 | 메커니즘 |
|---|---|---|
| F1 | → G1, F2 | 시장→PMF 전제, 고객→BM 제약 |
| F2→E2 | F3→G3 | BM→펀딩, 해자 없는 스케일=경쟁자 초대 |
| F4→F3 | G1→G2 | 기술→해자, 0→1→성장 전제 |
| G2 | → E2, E1 | 성장→펀딩·조직 확장 압력 |
| G3→E1 | G5→F3 | 스케일업 제약=조직, NW효과=최강 해자 |
| S1→F1 | S2→G4 | 피봇=시장 재선택, 파트너=국제화 |
| S3→E1 | E1→전체 | 다각화→복잡도↑, 실행력의 기반 |
| E2→E3 | E3→S1,S4 | 자본 고갈=생존 위기→피봇/퇴장 |
| E4→F1 | E5→S1,G1 | 규제→시장 변경, 타이밍→PMF |

## 사용법

전략 모드·판정 모드의 "축간 영향 체크" 단계에서 이 표를 참조하여, 핵심 축 결정이 유발하는 2차 효과를 전략 서술에 포함한다. 무시 시 전략이 고립된 선언에 그침.
