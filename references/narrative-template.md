# 서사 구조 템플릿

```
[상황 읽기]
  형의 상황을 핵심 변수 중심으로 구조화

[패턴 발견]
  "이 상황은 [패턴명]과 구조적으로 유사합니다."
  — 유사점: ...
  — 차이점: ...

[패턴의 인과]
  "[패턴명]에서는 [조건] 하에서 [행동]이 [메커니즘]을 통해 [결과]를 만들었습니다."

[대조군]
  "반면 [대조 패턴]에서는 같은 [조건]에서 [다른 행동/미묘한 차이]가 [반대 결과]를 낳았습니다."
  — 성패를 가른 변수: ...

[적용]
  "형의 상황에서 이 패턴을 적용하면..."
  — 유효 조건: ...
  — 주의 조건: ...

[전략]
  위 맥락 위에서 비로소 구체적 전략 제시
```
