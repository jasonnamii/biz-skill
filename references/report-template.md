# 비즈스킬 리포트 템플릿

§4.5 리포트 변환 시 이 구조로 재구성한다.

---

## 본문 — 3블록 구조

### 블록1: 진단 (상황 서사)

형의 상황이 왜 지금 이 구조인지를 서술한다.
핵심 변수·제약 조건·현재 위치를 하나의 이야기로 엮는다.
표가 필요하면 서술 뒤 보조로만 배치.

```
[작성 가이드]
- "형의 사업은 현재 ~한 상황이다. 이 구조가 형성된 이유는..."
- 업종·단계·규모·제약을 자연스럽게 서술 안에 녹인다
- 최소 3문장. 표·리스트 단독 사용 금지
```

### 블록2: 패턴 대조 (인과 서사)

매칭된 성공 패턴과 실패 패턴을 하나의 이야기로 엮는다.
패턴을 개별 나열하지 않고, 대비 구조로 통합 서술한다.

```
[작성 가이드]
- "[성공 사례]에서는 [조건] 하에서 [행동]이 [결과]를 만들었다.
   반면 [실패 사례]에서는 같은 조건에서 [다른 선택]이 [반대 결과]를 낳았다.
   성패를 가른 변수는 [X]이다."
- 성공/실패를 별도 섹션으로 분리하지 않는다 — 하나의 대비 서사로
- 최소 3문장. 패턴명·매칭 등급은 부록에
```

### 블록3: 전략 + 액션 (의사결정 가이드)

조건부 전략을 서술형으로 제시한다.
"만약 ~라면 ~가 유효하다. 단, ~이면 역효과가 난다."

```
[작성 가이드]
- 진단 모드: 블록3 생략 (블록1+블록2로 종결)
- 전략 모드: 전략 서술 → 유효/무효 조건 → 액션 플랜
- 판정 모드: 선택지별 서술 → 판단 기준 → 방향 제시 → 재평가 기한
- 최소 3문장. 조건부 형식("만약 ~라면") 필수
```

## 부록 (본문 하단 배치)

본문의 근거 데이터. 필요시 참조용.

### A. 스크리닝 결과
18축 ●/○/- 1줄 판정 원본.

### B. 매칭 근거
패턴별 유사도 등급(A/B+/B-/C) + 매칭 근거 1줄.

### C. 축간 영향 맵
핵심 축 간 영향 관계 (§5에서 해당 부분 추출).

### D. 리스크 매트릭스 (판정 모드만)
| 선택지 | 최선 | 최악 | 핵심 변수 | 되돌림 가능성 |

### E. 재평가 기한 (판정 모드만)
재평가 시점(3/6/12개월) + 트리거 변수.
