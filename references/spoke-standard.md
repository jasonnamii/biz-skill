# Spoke Standard — 18축 스포크 표준 구조

모든 스포크 파일(`f1-market.md`, `g2-growth.md` 등)이 지켜야 할 공통 구조.

## 필수 6섹션

1. **성공 패턴 라이브러리** — 동일 축에서 성공한 패턴의 조건·메커니즘·결과
2. **실패 패턴 라이브러리** — 동일 축에서 실패한 패턴의 조건·메커니즘·결과
3. **패턴 쌍 대조표** — 절대규칙 #3(대조군 필수) 구현. 성공·실패를 가른 핵심 변수 1열 병치
4. **진단 체크리스트** — 이 축이 현재 어떤 상태인지 판정하는 질문 목록
5. **조건부 전략 매트릭스** — 상태별 권장 전략 + 유효·무효 조건
6. **축간 연결** — 이 축이 다른 17개 축에 미치는 영향 + 다른 축이 이 축에 미치는 영향

## 규칙

- 명칭 변형 허용 (예: "성공 패턴" → "성공 사례", "케이스 라이브러리")
- 섹션 부재 불허 — 6개 모두 존재해야 함
- 섹션 순서는 권장이며, 콘텐츠 특성에 따라 재배열 가능
- 각 패턴에는 "조건 + 메커니즘 + 결과" 3요소 필수
